Ce projet constitue un formulaire d'inscription avec validation et tests, déployé sur Github Pages. Il contient les champs suivants : nom, prénom, email, date de naissance, ville, code postal.

Voici les commandes nécessaires pour lancer le projet : 

- npm install
- npm start

La commande "npm test" permet quant à elle de lancer les tests et de générer un rapport de couverture.

Avec une couverture de 100%, les tests permettent notamment de s'assurer du bon fonctionnement du calcul de l'âge, de la validation des champs, ou encore de la sauvegarde des informations dans le local storage.
